Installation Instructions

  1. Download the .tar.gz file using the button above
  2. Extract the file on your computer:
    tar -xzvf ai-recruiter-pro.tar.gz
  3. Create a new GitHub repository
  4. Push the files to your GitHub repository:
    cd ai-recruiter-pro
    git init
    git add .
    git commit -m "Initial commit"
    git remote add origin https://github.com/yourusername/ai-recruiter-pro.git
    git push -u origin main
  5. Follow the setup instructions in the README.md file to configure and run the application

Demo Account

The application includes a built-in demo admin account:

  • Email: demo@example.com
  • Password: Controlled by the DEMO_PASSWORD environment variable

For security, the demo account password is automatically synchronized with the DEMO_PASSWORD environment variable at application startup. Be sure to set this to a secure value in your production environment.
